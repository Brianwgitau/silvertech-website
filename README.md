# Silvertech Electrical

Silvertech Electrical is a serverless, Firebase-integrated web platform for showcasing professional electrical and telecom services, managing a premium store experience, and supporting secure authentication.

## Highlights
- Neon-metallic front-end design with responsive layout
- Firebase Authentication for protected access
- Firestore-backed service discovery and admin publishing flow
- Freemium store experience with public browse and gated download access

## Structure
- index.html: landing page
- pages/: auth, services, store, dashboard, admin views
- assets/css/style.css: shared visual styling
- js/: Firebase setup, authentication, services, and store logic
